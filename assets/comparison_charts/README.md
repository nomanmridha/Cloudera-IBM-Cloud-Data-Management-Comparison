# Comparison Charts

Use this file to store chart images or Markdown tables showing key differences.

## Chart Types:
- Feature comparison table (e.g., AI/ML, governance, hybrid support)
- Deployment model comparison
- Use case suitability

Example:
| Feature | Cloudera | IBM Cloud |
|---------|----------|-----------|
| AI/ML Integration | Supported | Strong |
| Governance & Metadata | SDX | Knowledge Catalog |
| Managed Service | Moderate | Excellent |
